# Sales and profit Dashboard (Excel Project)

## Project Overview
This project is a Sales Dashboard created using Microsoft Excel for practice. The dashboard helps analyze sales performance by visualizing important metrics such as total sales, profit, customers, regional performance, and top cities.

## Key Metrics
- Total Customers: 1130  
- Total Orders: 1952  
- Total Sales: $1,924,337.8  
- Total Profit: $224,077.61  

## Dashboard Insights
- The East region generates the highest profit among all regions.
- The South region has the lowest profit.
- Profit shows a strong increase after March.
- May records the highest profit among the months shown.
- Office Supplies contribute the highest share of profit compared to Furniture and Technology.
- Cities like Washington, Thornton, and New City are among the top contributors to profit.

## Features
- Region-wise profit analysis
- Month-wise profit trend
- Category-wise profit distribution
- Top 10 cities by profit
- Interactive filters for Region and Month

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Purpose
The purpose of this project is to practice creating dashboards and visualizing data using Excel.
